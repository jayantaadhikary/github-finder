## A github user finder project using the Github api created using ReactJS.

### Do make sure to copy the github tokkens by creating one in your settings (settings > developer settings > personal tokken). Then after cloning+forking the project to your system, run `npm install`  to install all the files required for the project. Also make sure to create a .env file where you put your github tokken and api link. Then run the project using `npm start`
